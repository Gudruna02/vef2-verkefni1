1. Hvað gera eftirfarandi Linux skipanir?
cd - change directory, færir sig um möppu.
ls - list, sjá hvaða verkefni eru í möppuni.
pwd - print working directory, prenta hvaða möppu þú ert í
mkdir - make directory, búa til nýja möppu

2. Hvað gera eftirfarandi Git skipanir?
git clone - búa til nákvæma eftirlíkingu 
git status - sýnir hvaða möppur þú hefur breytt og þær sem þú átt eftir að commita
git diff - sér breytingarnar

3. Hvað gera eftirfarandi Git skipanir saman? Hvaða gagn er að þeirri aðgerð?
git log - Sýnir lista af öllum commits
git checkout - skiptir yfir í tilgreinda grein 
git branch - Skrifar lista af öllum greinum

4. Hvað er útgáfustýring (Version Control)?
Útgáfustýring er að geta fengið fyrri útgáfu á skjali.

5. Hverjir eru helstu kostir við að nota GIT?
það er þæginlegt að savea skjöl og verkefni í því án þess að það eyðist/týnist.
þetta er á netinu svo þu getur opnað skjölin/verkefnin hvar sem er.  
það er auðvelt að nota Git en ekki Git bash

6. Hversu oft telur þú að eigi að gera færslur (commit) í verkefni?
Við hverja svaraða lausn (fer eftir hversu löng/erfið hún er).

7. Hvað er átt við með “Working Directory”?” og “Staging Area” í GIT?
Working directory - Skráin sem geymir allt sem er commitað
Stagin area - þar sem þú getur testað allt áður en þú commitar
